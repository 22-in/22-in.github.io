Privacy Policy for QuickCode

Last Updated: March 21, 2026

1. Introduction

Welcome to QuickCode, developed by 22. We are committed to protecting your privacy while providing a powerful mobile development environment for Python and Kivy/KivyMD.

2. Information We Collect

  • User Code & Projects: All code created within QuickCode is stored locally on your device unless you explicitly choose to export or share it.
  • Technical Data: We may collect basic device information (OS version, model) and crash logs to optimize the IDE's performance.
  • API Integrations: If you use AI features via the Gemini API, certain prompts may be processed to provide responses.

3. Key Permissions

To function as an IDE, QuickCode requires:

  • Storage (Read/Write): To create, save, and manage your Python (.py) files and project assets.
  • Internet Access: To download necessary libraries, use the Gemini API, and provide updates.
  • Microphone/Audio: (Only if your specific project uses PyAudio or similar features).

4. Third-Party Services

We use the following third-party services:

  • Google Gemini API: For AI-assisted coding features. Please note that your interactions with the AI are subject to Google’s Generative AI terms.
  • Python Libraries: Standard open-source libraries used for the execution environment.

5. Data Security

As a developer tool, we prioritize your intellectual property. We do not claim ownership of the code you write in QuickCode. We recommend users manage their own sensitive data (like private API keys) carefully within their scripts.

6. Children's Privacy

QuickCode is a technical tool. While it can be used for learning, we do not knowingly collect personal data from children under the age of 13.

We may update this policy as we add new features (like cloud sync or new frameworks). We recommend checking this document periodically.
